## Problem Statement

As block subsidies decline, Bitcoin must increasingly rely on transaction fees. Current fee market assumptions may not sufficiently support long-term miner decentralization.

This repository explores economic modeling and protocol-level considerations to address:

- Miner revenue stability
- Transaction velocity constraints
- Custodial concentration effects
- Fee market mathematical limitations
